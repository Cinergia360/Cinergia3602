Repositorio de mediación

Proyectos, películas y recursos

Explora los universos narrativos trabajados en Cinergia 360. Cada proyecto puede incluir una ficha de proyecto y/o un conjunto de recursos de mediación (materiales de consulta, video, modelos 3D o apoyos accesibles).

Cómo usar este repositorio

  • Ver ficha: información general del proyecto, enfoque y contexto de mediación.
  • Ver recursos: materiales disponibles para consulta o exploración (video, modelos 3D, audiodescripción).
  • Estado: algunos proyectos están en proceso y pueden aparecer como Próximamente.

Leyenda rápida

Disponible — el proyecto tiene recursos para consultar.
Próximamente — el proyecto está en proceso de desarrollo.
